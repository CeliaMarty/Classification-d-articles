# Classification-d-articles

Ce projet contient le code pour classifier des articles de presse en différentes catégories (Sport, Business, Tech, Entertainment).  

Le dossier inclut :  
Préparation et nettoyage du dataset project_dataset_bbc_text.csv  
Analyse exploratoire et visualisation des articles  
Modèle de Machine Learning (Naive Bayes avec TF-IDF et GridSearchCV)  
Modèle de Deep Learning (LSTM avec Keras)  
Évaluation et comparaison des modèles  
Prédictions sur de nouveaux articles  

Le code est prêt à être exécuté dans Google Colab ou localement.
